Sebelum memulai

Selamat datang di WordPress. Sebelum memulai, Anda harus mengetahui hal-hal berikut.

  1. Nama basis data
  2. Nama pengguna basis data
  3. Sandi database
  4. Host basis data
  5. Prefiks Tabel (jika Anda ingin menjalankan lebih dari satu WordPress pada satu basis data)

Informasi berikut dibutuhkan untuk membuat berkas wp-config.php. Tidak perlu khawatir jika proses pembuatan file otomatis gagal disebabkan oleh hal tertentu. Proses ini hanya untuk mengisikan informasi basis data ke dalam file konfigurasi. Anda juga bisa membuka file wp-config-sample.php dengan sebuah editor teks, lengkapi informasi, kemudian simpan sebagai wp-config.php. butuh lebih banyak bantuan? Baca artikel dukungan di wp-config.php.
